### O que aprendi

Nesse projeto envolvendo HTML, CSS e JavaScript, aprendi conceitos importantes. Enquanto o HTML cria a estrutura da página e o CSS cuida do visual, o JavaScript é a linguagem de programação que dá vida e inteligência à aplicação.

Neste projeto de formulário, o JS desempenha um papel crucial na experiência do usuário e na garantia de qualidade (QA) dos dados, sendo responsável por:

* Validação em Tempo Real: Ele verifica instantaneamente se os dados inseridos estão corretos (como o formato de um e-mail ou a quantidade de caracteres de uma senha), sem a necessidade de recarregar a página.

* Manipulação do DOM: O JavaScript interage diretamente com os elementos da tela, exibindo mensagens de erro claras ou alertas de sucesso de forma dinâmica assim que o usuário preenche um campo.

* Prevenção de Falhas: Atuando como a primeira linha de defesa, o script impede que formulários em branco ou com informações inválidas sejam enviados, poupando processamento e evitando dores de cabeça no futuro.
